Resumen y servicios

Resumen Ejecutivo

FiscalPoint permite a la empresa operar facturación electrónica e-CF con cumplimiento DGII, reducción de trabajo manual y trazabilidad de cada comprobante. La solución cubre emisión, recepción, XML firmado, representación impresa, consulta de estados, aprobación comercial y soporte operativo.

Beneficio fiscal

Cumplimiento con la normativa de facturación electrónica en República Dominicana.

Beneficio operativo

Menos tareas manuales y mayor velocidad para facturar, consultar y entregar documentos.

Beneficio de control

Información centralizada para ventas, gastos, clientes, proveedores y documentos recibidos.

Requisitos del cliente

Información Que Debe Proveer el Cliente

Para evitar retrasos, el cliente debe entregar estos elementos antes de iniciar la configuración técnica y el proceso de certificación DGII. Esta información es responsabilidad del cliente y es necesaria para firmar, certificar y operar correctamente.

1
Certificado digital vigente Archivo de Firma Digital del contribuyente y contraseña correspondiente para firmar XML e-CF.
2
Datos fiscales del emisor RNC, razón social, nombre comercial, dirección, municipio, provincia, teléfono, correo y actividad económica.
3
Formato de factura Diseño o layout actual de factura para preparar la representación impresa que verá el cliente final.
4
Información para DGII Datos, formularios, accesos o documentos requeridos durante el proceso oficial de certificación.
5
Secuencias y tipos e-CF Tipos de comprobantes que utilizará la empresa y autorizaciones de secuencias cuando correspondan.
6
Responsable del proyecto Persona de contacto administrativa o contable para validar pruebas, formatos y decisiones operativas.

Si Habrá Migración de Datos

El cliente debe entregar un backup con:

  • Clientes.
  • Productos y servicios.
  • Facturas históricas.
  • Suplidores y proveedores.
  • Cualquier otro dato relevante para operar.

Validación previa

La migración depende de la calidad y estructura del respaldo entregado. Antes de importar datos definitivos se revisa una muestra, se confirman campos obligatorios y se acuerda el alcance exacto.
